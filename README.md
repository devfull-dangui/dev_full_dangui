# 🚀 Meu Portfólio Pessoal - Deborah Dangui

Este é o meu primeiro projeto de portfólio totalmente responsivo, desenvolvido para centralizar minhas redes sociais e praticar conceitos avançados de Front e Back-end.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica.
- **CSS3**: Layout com Flexbox, Media Queries e animações personalizadas.
- **JavaScript**: Lógica de redirecionamento e manipulação do DOM.
- **Font Awesome**: Ícones modernos.

## 📱 Funcionalidades

- [x] Design responsivo (ajusta em celulares e computadores).
- [x] Animações de entrada (Fade In).
- [x] Rodapé fixo (Sticky Footer).
- [x] Links integrados com abertura em nova guia.
